The default value will appear as an initial value when mobile users open the form. There are some question type specific properties: Use mobile user's username as the default value: this will put the mobile user's ProntoForms username as an answer to the question. Note: If a question is Required but has a default answer, it is considered Answered in the app. This means that mobile users can skip answering the question themselves.
